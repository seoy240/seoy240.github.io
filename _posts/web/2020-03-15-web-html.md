### html tag

* `<strong></strong>` :  bold
* `<u></u>` : underline
* `<h1></h1>`: heading(h1-h6)
* `<br>` : enter
	* 자유롭게 줄바꿈 가능
* `<p></P>` : paragraph
	* 단락을 표현할때 좋지만 단락과 단락의 간격이 고정되어 있어 시각적 자유도 떨어짐.
	* CSS를 통해 극복 가능
* `<img>`  : 이미지
* `<ul></ul>`/`<ol></ol>`: unordered list / ordered list
	* `<li></li>` : list
* `<table></table>`
	* `<tr></tr>`
		* `<td></td>`
* `<head></head>` : 본문 설명
	* `<title></title>` :제목
	* `<meta></meta>` : 본문 속성
* `<body></body>` : 본문 내용
* `<div></div>` : division - 가상의 레이아웃을 나누는데 사용
	* 주로 CSS wrapping 용도
	* display 속성 : block (줄바꿈됨)
* `<span></span>` : div 태그와 마찬가지로 가상의 레이아웃을 나누는데 사용
	* 주로 CSS wrapping 용도
	* display 속성 : inline (줄바꿈 되지 않음)
