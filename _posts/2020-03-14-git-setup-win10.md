## Git installation
[Git](https://git-scm.com/downloads)에서 win 10용 git을 설치한다.

## User configuration
설치후 git bash를 열어서 user configuration을 해준다.

```
$ git config --global user.name "user_name"
$ git config --global user.email "user_email@example.com"
```
